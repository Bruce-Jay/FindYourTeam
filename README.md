# 项目介绍

该项目使用 vue2 + elementUI 进行开发，并使用 yarn 作为包管理工具。

## 项目设置
```
yarn install
```

### 编译并以热重载方式进行开发
```
yarn run serve
```

### 编译并为生产环境压缩
```
yarn run build
```

### 自定义配置
参见 [配置参考](https://cli.vuejs.org/config/)。
